## How do I use this?

First you need to have Java 8 or above installed. Yeah, Java. Deal with it.

Second you need an image that has transparency. You can do this in Photoshop or GIMP yourself (I'm not going to teach you how). Basically, the transparent pixels will be blank and the others will have whatever character you picked (the default is the colon `:`). You MUST save the image as a **.png** with NO compression. I don't know why but Java complains if it has compression, at least on my machine. Save the image in the same folder as the `.jar` file.

Third, fire up your command processor (command prompt on Windows, Terminal on Linux or Mac). Navigate to the folder where the `.jar` is (using the `cd` command. On Windows, you can also click the navigation bar in Explorer and type `cmd` and press Enter; it will open a command window in that folder). The `.jar` is located in the `bin/` folder but you can copy it wherever.

Fourth you need to type the command to run the program:

`java -jar ColonConvert.jar filename.png <options>`

But hold on!! You have more options to pick from:

    -o=output.txt        When this is used, the output is saved to a file. If not then it is printed in the command
                         Window. You should use this. Replace output.txt with whatever filename you want.
                         
    -c=B                 This changes the character that is used. Default is the : but the # looks good too.
                         
    -s=3                 This changes the scale value. Default is 2. Replace the 3 with whatever scale you want. Be careful with 1, it can take a long time!

When you have all the options all nice and figured out, smash that Enter button, ~~hit like, leave a comment, get subscribed and click the bell to get notifications~~. After it finishes it will either print the output or save the text to a file. If it fails, you probably went wrong somewhere.

You can now post the text to Reddit or Instagram (Instagram limits the amount of text in a message so you might not get the whole thing). **IMPORTANT** If you post to Reddit: put the text in a code block. This is done putting three \` on a line, like \`\`\`. Then on the next line, paste the text. End it with another three \`.

That's it!

## It didn't work or it took too long...

Your computer is probably too slow for the scale value you picked. The default is 2. If the image you are using is pretty big, change the scale value! 5 or 10 make good outputs for larger images. If it gives an error and you can't figure it out yourself (and you didn't modify my code), then create an Issue and I might help depending on your knowledge of the English language (I am more likely to help more entertaining Issue's that still use good English, i.e. ***MORE MEMES***).
